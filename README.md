# 探索豆 ESP32 固件

MicroPython 固件，用于探索豆物联网教育平台。

## 📦 固件下载

最新固件会自动构建并发布到 [Releases](https://github.com/dakeqi/tansuodou-firmware/releases)

## 🔧 支持的硬件

- ESP32
- ESP32-S2
- ESP32-S3  
- ESP32-C3

## 📖 使用说明

访问 [探索豆物联平台](https://tansuodou.com) 进行在线烧录和配置。

## 🚀 自动构建

固件由 GitHub Actions 自动构建，每次更新会自动发布新版本。
